## 🌍 English

**FruitGuard System** is an advanced, AI-powered multi-platform application designed for fruit quality analysis and disease diagnosis. This project leverages Deep Learning models to provide accurate assessments across various platforms.

### 🚀 Features
- **AI-Powered Analysis**: High-accuracy models for fruit detection and health diagnosis.
- **Multi-Platform Support**:
  - **Mobile**: Cross-platform app built with Flutter.
  - **Desktop**: Native Windows application built with PyQt5.
  - **Backend**: High-performance API built with FastAPI.
- **Robust API**: Scalable backend with authentication and real-time analysis endpoints.

### 🛠️ Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, SQLite.
- **Mobile**: Flutter, Dart.
- **Desktop**: Python, PyQt5.
- **AI/ML**: TensorFlow/Keras or PyTorch (depending on the models used).

### ⚙️ Setup Instructions
1. **Backend**:
   - Navigate to `backend/`.
   - Install dependencies: `pip install -r requirements.txt`.
   - Run the server: `python run_backend.bat` or `uvicorn app.main:app`.
2. **Mobile**:
   - Navigate to `frontend/`.
   - Run `flutter pub get`.
   - Build/Run: `flutter run`.
3. **Desktop**:
   - Navigate to `pyqt_desktop/`.
   - Run `python main.py` (or use `run_pyqt.bat`).
